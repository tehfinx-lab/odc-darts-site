# ODC Online Darts Circuit Website

Free modern darts league website starter.

## What this includes

- Home overview
- Review cards
- Tables page with dropdown
- Player stats
- Leaderboards
- Events / nightly tournament winners
- Mobile-friendly dark UI
- Your ODC logo included

## How to run locally

```bash
npm install
npm run dev
```

## How to publish free on Vercel

1. Upload this folder to GitHub
2. Go to Vercel
3. Click Add New Project
4. Import the GitHub repo
5. Click Deploy
